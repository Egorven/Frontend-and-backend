Отчёт о реализации практик
Тема проекта: Создание магазина музыкальных инструментов

Практическое занятие №1 CSS-препроцессоры

Расположение: src/pages/ProductPages/ProductsPage.scss
SCSS-стили
Для чего испоьзуется в проекте:
1.Стилизация страницы товаров
2.Стилизация форм и модальных окон
3.Удобство поддержки, переиспользование стилей

Практическое занятие №2 Сервер на Node.js + EXpress

Расположение:server.js
1.Express-сервер - Базовая настройка сервера
2.Middleware - Обработка запросов, логирование
3.Маршрутизация - CRUD-эндпоинты для товаров и категорий

Практическое занятие №3 JSON и внешние API
1.JSON-структуры данных - server.js - Хранение категорий и товаров в памяти
2.Axios-клиент для API - src/api/index.js - HTTP-запросы с фронтенда на бэкенд
3.Валидация JSON-данных - server.js - Проверка обязательных полей перед сохранением

Практическое занятие №4 API + React
1.Интеграция с API - src/api/index.js - Единый клиент для всех запросов
2.CRUD из интерфейса - src/pages/ProductPages/ProductsPage.jsx - Создание, чтение, обновление, удаление товаров
3.Состояние (useState) - ProductsPage.jsx - Хранение списка товаров и категорий
4.Формы и модальные окна - ProductModal.jsx - Ввод данных пользователем

Практическое занятие №5 REST API(Swagger)
Расположение:server.js

1.Swagger-конфигурация - Настройка OpenAPI спецификации
2.Swagger UI - Интерактивная документация
3.Документация всех CRUD - Полное описание API для разработчиков
