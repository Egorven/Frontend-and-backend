Тема проекта: Создание PWA-приложения для управления заметками/задачами

Практическое занятие №13: Service Worker
Расположение:
index.html, app.js, sw.js
Для чего используется в проекте:

- Кэширование статических ресурсов (HTML, CSS, JS) для работы приложения без интернета
- Регистрация Service Worker
- Реализация стратегии кэширования "Cache First" для статических файлов
- Обеспечение базовой офлайн-работоспособности интерфейса

Практическое занятие №14: Web App Manifest
Расположение:
manifest.json, icons/, index.html, sw.js
Для чего используется в проекте:

- Создание JSON-манифеста с метаданными приложения (name, short_name, start_url, display: standalone, цвета, иконки)
- Подготовка набора иконок разных размеров (16x16 – 512x512) для корректного отображения на разных устройствах
- Возможность установки приложения на домашний экран и запуска в изолированном окне без интерфейса браузера

Практическое занятие №15: HTTPS + App Shell
Расположение:
localhost+2.pem, localhost+2-key.pem, index.html, app.js, sw.js, content/
Для чего используется в проекте:

- Настройка локального HTTPS-сервера (через `mkcert`) для корректной работы Service Worker и современных PWA-функций
- Реализация архитектуры App Shell: мгновенная отрисовка базового каркаса приложения (шапка, навигация, основной контейнер)
- Разделение кэшей на статический (`CACHE_NAME`) и динамический (`DYNAMIC_CACHE_NAME`)
- Стратегия "Network First" с fallback на кэш  для динамических страниц при отсутствии сети

Практическое занятие №16: WebSocket + Push
Расположение:
Backend: server.js, Frontend: index.html, app.js, sw.js
Для чего используется в проекте:

- Генерация VAPID-ключей для безопасной идентификации сервера
- Управление подписками на push-уведомления
- Обработка отображения системных уведомлений при закрытой вкладке

Практическое занятие №17: Детализация Push (Напоминания и откладывание)
Расположение:
Backend: server.js, Frontend: app.js, sw.js, content/home.html
Для чего используется в проекте:

- Расширение формы создания заметок: добавление поля выбора даты и времени напоминания
- Реализация отложенной отправки push-уведомлений в заданное время
- Добавление действия «Отложить на 5 минут»
- Обеспечение работы системы напоминаний в фоне: уведомления приходят  при закрытой вкладке
